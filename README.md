# gunheeShin.github.io

Personal website of Gunhee Shin — [gunheeshin.github.io](https://gunheeshin.github.io).

Built with [Jekyll](https://jekyllrb.com/) on the
[Beautiful Jekyll](https://beautifuljekyll.com/) theme by Dean Attali.

## Local development

```bash
bundle install
bundle exec jekyll serve
```

Then open <http://localhost:4000>.

## CV

The CV is edited as HTML and built to PDF with headless Chrome:

```bash
./cv/build.sh   # cv/cv.html -> cv/cv.pdf
```
